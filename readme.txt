Display a custom number of recent results from All Elite Wrestling matches or from a particular wrestler.

== Description ==

You can now display results from All Elite Wrestling events from all shows like Dynamite, Rampage, Collision, PPVs and others.

You also have the option to just display recent recents from a particular wrestler, handy if you have a wrestlers profile on your website.

Give your visitors the ability to quickly browse through recent results.

== Installation ==

1. Upload the `results-of-aew` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a post or page and enter [aewresults count='4'] and adjust the number to a max of 100
4. To display a wrestlers recent matches only use [aewresults count='7' wrestler='mariah-may']
